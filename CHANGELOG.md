# Changelog

All notable changes to PDF Editor are documented here.

## [Unreleased]

## [1.0.0] - 2025-03-17

### Added
- Initial release
- PDF viewer powered by Syncfusion Flutter PdfViewer
- Text annotation tool with bold/italic/font size support
- Highlight, underline, strikethrough markup tools
- Freehand drawing tool with colour and stroke width
- Rectangle and circle shape tools
- Eraser tool for removing individual annotations
- Colour picker with alpha/opacity support
- Stroke width slider
- Font size control
- Zoom in/out toolbar controls
- Collapsible page thumbnails panel
- Collapsible properties panel
- Collapsible annotations list panel
- Page navigator (first/prev/next/last)
- Save annotated PDF to device documents directory
- Share PDF via system share sheet
- Material 3 design with light and dark mode
- Android APK (minSdk 23, split per ABI)
- Android App Bundle for Google Play
- Windows 10+ desktop EXE
- NSIS Windows installer
- GitHub Actions CI/CD pipeline
